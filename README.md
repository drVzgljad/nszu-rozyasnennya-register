# Реєстр роз'яснень НСЗУ

Приватний робочий реєстр документів, зібраних з офіційного архіву НСЗУ.

## Вміст

- `03_За_напрямами/` - один структурований комплект документів за напрямами.
- `04_Реєстр/` - фінальний реєстр, контрольні таблиці та скрипти обробки.
- `05_Веб_реєстр/` - локальна вебсторінка з пошуком, фільтрами, реквізитами документа і пов'язаними файлами.
- `index.html` - точка входу до вебреєстру.

У вебреєстрі доступні фільтри за напрямом, пакетом, роком, датою та номером документа, форматом і станом текстового шару.

## Локальний перегляд

Через обмеження браузера сторінку потрібно відкривати через локальний HTTP-сервер, а не подвійним кліком по HTML-файлу.

```powershell
python -m http.server 8042
```

Після цього відкрийте `http://127.0.0.1:8042/index.html`.
