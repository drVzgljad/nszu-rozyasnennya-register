# Реєстр роз'яснень НСЗУ

Джерело: https://nszu.gov.ua/arxiv-dokumentiv?groups%5B2%5D%5Battributes%5D%5B%5D=36

Принцип робочих назв:

`Напрям_Пакет_Зміст_Рік.ext`

Оригінальна назва файла з сайту, URL завантаження та запропонована змістовна
назва фіксуються у реєстрі. До завершення аналізу завантажені оригінали не
перейменовуються.
