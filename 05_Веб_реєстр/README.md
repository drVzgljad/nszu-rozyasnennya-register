# Локальний реєстр роз'яснень НСЗУ

Для відкриття каталогу двічі натисніть файл `Відкрити_реєстр.cmd` у батьківській
папці `Робоча_розяснення_НСЗУ`.

У каталозі можна:

- шукати за назвою, пакетом або словами з тексту документа;
- фільтрувати документи за напрямом, роком, форматом і наявністю OCR;
- переглядати картку документа, змістовну та оригінальну технічну назву;
- відкривати локальну копію або офіційне джерело НСЗУ;
- переходити до пов'язаних документів і додатків.
- переходити до сторінки `Пакети 2026` з повнотекстовим пошуком по умовах закупівлі.
- переходити до сторінки `Постанова 1808` з тарифами, коефіцієнтами, строками та зв'язками до пакетів.

Файл `data/documents.json` створюється з фінального реєстру та локальних
текстових витягів за допомогою `build_web_data.py`.

Сторінка `pakety/index.html` читає структурований індекс
`pakety/data/packages_2026.json`, який створюється зі DOCX-пакетів скриптом
`pakety/build_packages_data.py`.

Сторінка `postanova/index.html` читає індекс
`postanova/data/resolution_1808.json`, який створюється з PDF постанови скриптом
`postanova/build_resolution_data.py`. Контрольовані прив'язки тарифних глав і
додатків до пакетів зберігаються у `postanova/data/package_resolution_links.json`.
